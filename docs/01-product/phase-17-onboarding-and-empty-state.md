# Phase 17: Onboarding & Empty State Experience

## Goal

Guide new users through initial setup with a multi-step onboarding form and clear empty state guidance.

## Deliverables

- Multi-step onboarding form for first-time/empty config
- Feed type selection (TV/movie) determines setup flow
- Guidance and validation for each step
- No sample data or suggestions

## Explicit Deferrals

- No onboarding for users unfamiliar with RSS/Transmission
- No sample feeds/shows/movies
